**三个线程循环输出ABCABCABC**

部分代码:
```Java
private AtomicInteger num = new AtomicInteger(0);
class A implements Runnable {
    @Override
    public void run() {
        while (num.get() < 10) {
            if (num.get() % 3 == 0 && num.get() < 10) {
                System.out.println(Thread.currentThread().getName() + ":" + num + ":" + "A");
                num.incrementAndGet();
            }
        }
    }
}
```
